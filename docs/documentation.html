Phase 1: Ground Truth Generation

Quick Start

  • Run the Phase 1 script
  • Wait for all subjects to process
  • View results in JSON file
  • Load results on website dashboard

Configuration

  • Subjects: 109 (PhysionetMI)
  • Cross-validation: 5-fold
  • CSP components: 4
  • Frequency band: 7-30 Hz

Output

  • JSON file with all results
  • Per-subject accuracy scores
  • Statistical summaries
  • Metadata and parameters

Usage Example

from src.generate_ground_truth_labels import PhysionetMIGroundTruthGenerator

# Create generator instance
generator = PhysionetMIGroundTruthGenerator(
    n_subjects=109,
    n_folds=5,
    random_state=42
)

# Generate labels for all subjects
results = generator.generate_all_labels(
    output_path='results/ground_truth_labels.json'
)

API Reference

PhysionetMIGroundTruthGenerator

Main class for generating ground truth labels from PhysionetMI dataset.

Parameters

  • n_subjects (int): Number of subjects to process (default: 109)
  • n_folds (int): Number of cross-validation folds (default: 5)
  • random_state (int): Random seed for reproducibility (default: 42)

Methods

  • load_subject_data(subject_id): Load and preprocess EEG data for a subject
  • build_decoder(): Create CSP + LDA pipeline
  • evaluate_subject(subject_id): Evaluate decoder performance for one subject
  • generate_all_labels(output_path): Process all subjects and save results

Returns

JSON file containing:

  • metadata: Configuration and parameters used
  • summary: Aggregate statistics across all subjects
  • subjects: Array of per-subject results with accuracy metrics

FAQ

Frequently Asked Questions

Q: How long does Phase 1 take to run?

A: Approximately 1-2 hours for all 109 subjects, depending on your hardware.

Q: Where is the dataset downloaded?

A: MNE automatically downloads PhysionetMI to ~/mne_data/ (approximately 2.5 GB).

Q: Can I process a subset of subjects?

A: Yes, modify the n_subjects parameter when creating the generator instance.

Q: What if some subjects fail to process?

A: The script handles errors gracefully and continues processing. Failed subjects are marked in the output JSON.

Q: How do I visualize the results?

A: Use the Results dashboard on this website, or load the JSON file with pandas/matplotlib.

Troubleshooting

Installation Issues

  • Use Python 3.8 or higher
  • Create a virtual environment
  • Update pip: pip install --upgrade pip
  • Install dependencies one by one if needed

Data Loading Errors

  • Check internet connection for dataset download
  • Ensure sufficient disk space (~3 GB)
  • Verify MNE data directory permissions
  • Clear MNE cache if corrupted

Memory Issues

  • Process fewer subjects at a time
  • Close other applications
  • Monitor system resources
  • Consider using a machine with more RAM
