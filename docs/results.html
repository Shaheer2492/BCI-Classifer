How to Generate Results

Run Phase 1 Script

# Navigate to the project directory
cd /home/user/BCI-Classifer

# Install dependencies (first time only)
pip install -r requirements.txt

# Run ground truth label generation
python src/generate_ground_truth_labels.py

# Results will be saved to results/ground_truth_labels.json
Note: The first run will download the PhysionetMI dataset (~2.5 GB). Total processing time: approximately 1-2 hours for all 109 subjects.
