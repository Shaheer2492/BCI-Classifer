Project Overview

This project develops a machine learning system to predict Brain-Computer Interface (BCI) decoder performance from early motor imagery trials. By identifying strong and weak performers early in the training process, we can optimize BCI calibration and improve user experience.

Key Features

Dataset

  • PhysionetMI: 109 subjects
  • Motor imagery tasks
  • Left/right hand, feet
  • 64-channel EEG

Decoder

  • CSP (4 components)
  • Linear Discriminant Analysis
  • 5-fold cross-validation
  • 7-30 Hz bandpass filter

Features

  • Band power (C3/C4/Cz)
  • Spatial filters (CSP)
  • ERD/ERS magnitude
  • Trial variability & SNR
