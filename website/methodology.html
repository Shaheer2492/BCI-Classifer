Methodology

Four-phase approach to BCI performance prediction

Phase 1: Ground Truth Label Generation

Establish baseline performance metrics for each subject using MetaBCI decoders.

Dataset

  • PhysionetMI dataset
  • 109 subjects
  • Motor imagery tasks (runs 4, 8, 12)
  • Left/right hand, hands/feet
  • 64-channel EEG recordings

Decoder Configuration

  • Common Spatial Patterns (4 components)
  • Linear Discriminant Analysis
  • Bandpass filter: 7-30 Hz
  • Epoch window: 1-4 seconds

Cross-Validation

  • Within-subject evaluation
  • 5-fold stratified CV
  • Maintains class balance
  • 80/20 train-test split per fold
Rationale: Within-subject CV is appropriate for Phase 1 because we aim to measure each individual's decoder performance, not cross-subject generalization. This provides reliable ground truth labels for the prediction model.
